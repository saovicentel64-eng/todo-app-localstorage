# 📝 To-Do List App com Local Storage

Uma aplicação moderna e produtiva de gerenciamento de tarefas com armazenamento local.

## ✨ Características

✅ **Local Storage** - Dados salvos automaticamente  
✅ **Temas Claro/Escuro** - Alterne entre temas  
✅ **Filtros** - Todas, Ativas, Concluídas, Favoritas  
✅ **Prioridades** - Alta, Média, Baixa  
✅ **Estatísticas** - Veja seu progresso  
✅ **Responsivo** - Funciona em todos os dispositivos  

## 🚀 Começar

1. Clone: `git clone https://github.com/saovicentel64-eng/todo-app-localstorage.git`
2. Abra `index.html` no navegador
3. Comece a adicionar tarefas! 📝

## ⌨️ Atalhos

- `Enter` - Adicionar tarefa
- `Ctrl+S` - Exportar
- `Ctrl+L` - Limpar tudo

## 📊 Tecnologias

- HTML5
- CSS3 (Tailwind + Custom)
- JavaScript Vanilla
- Local Storage API
